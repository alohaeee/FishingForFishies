# FishingForFishies
Yolo-coded game with Component object model.

## Gameplay
Small and quite simple arcade. The main point of game is to catch fishies as much as possible.
## Control
Here is player control help-note
```
A,D - to move boat;
SPACE - to throw hook;
UP,DOWN - to move hook;
```
